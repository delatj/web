TU LÓGICA DE PROGRAMACIÓN
ESCUELA DE DIAGRAMAS
1. Terminal (Óvalo)

Es el botón de Encendido y Apagado. Todo algoritmo debe tener un principio y un final claro.

2. Proceso (Rectángulo)

Son las órdenes que el motor ejecuta. Si escribes "avanzar", el ordenador crea este bloque.

3. Decisión (Rombo)

Es la pregunta lógica. Tiene dos salidas: Si es SÍ, avanzamos. Si es NO, activamos el bucle.

4. Bucle (Línea)

Es una repetición. Si fallas la decisión, esta línea te obliga a volver arriba para intentarlo de nuevo.
