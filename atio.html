DESCRIPCIÓN DE LOS MATERIALES

1.1 Materiales de medición

Probeta (Graduated Cylinder)

Instrumento de laboratorio utilizado para medir volúmenes de líquidos con mayor precisión que un vaso de precipitados. Tiene forma cilíndrica y una base para mantenerla estable.

Bureta (Burette)

Instrumento de laboratorio utilizado en volumetría para medir con precisión el volumen de líquido que se vierte. Se usa principalmente en titulaciones.

Termómetro (Thermometer)

Instrumento utilizado para medir la temperatura de sustancias. En laboratorio suele ser de vidrio y contiene mercurio o alcohol coloreado.

Cuentagotas (Dropper)

Instrumento utilizado para transferir pequeñas cantidades de líquido. Consiste en un tubo con una punta cónica y una perilla de goma para succionar.

Calibre (Caliper)

Instrumento de medición utilizado para medir dimensiones de objetos pequeños con gran precisión. Puede medir diámetros internos, externos y profundidades.

Metro (Tape Measure)

Instrumento de medición utilizado para medir longitudes. En laboratorio se utilizan metros más pequeños y precisos para medir objetos.

Báscula (Scale)

Instrumento utilizado para medir la masa de objetos o sustancias. En laboratorio se utilizan básculas de precisión para obtener medidas exactas.

Cronómetro (Stopwatch)

Instrumento utilizado para medir intervalos de tiempo con gran precisión. Es esencial en experimentos que requieren mediciones temporales exactas.

1.2 Materiales de almacenamiento

Vaso de precipitado (Beaker)

Recipiente cilíndrico con fondo plano utilizado para contener, mezclar o calentar líquidos. Tiene una boca ancha que facilita el vertido.

Vidrio de reloj (Watch Glass)

Pequeña pieza de vidrio cóncava utilizada para evaporar líquidos, pesar pequeñas cantidades de sólidos o cubrir vasos de precipitados.

Tubo de ensayo (Test Tube)

Pequeño tubo de vidrio utilizado para contener pequeñas muestras de líquidos o sólidos. Se utiliza en experimentos y reacciones químicas.

Matraz de fondo redondo (Round-bottom Flask)

Recipiente de vidrio con fondo esférico utilizado para calentar líquidos de manera uniforme. Se usa en destilaciones y reacciones químicas.

Matraz Erlenmeyer (Erlenmeyer Flask)

Recipiente cónico con base plana utilizado para mezclar o contener líquidos. Su forma evita pérdidas por evaporación y facilita la agitación.

Matraz aforado (Volumetric Flask)

Recipiente de vidrio con cuello largo y una marca que indica un volumen específico. Se utiliza para preparar soluciones de concentración exacta.

Cristalizador (Crystallizing Dish)

Recipiente de vidrio de fondo plano y paredes bajas utilizado para cristalizar sustancias o como recipiente en baños maría.

Frasco lavador (Wash Bottle)

Recipiente de plástico con un tubo curvo utilizado para enjuagar recipientes o añadir pequeñas cantidades de agua destilada.

1.3 Materiales de soporte o manipulación

Espátula (Spatula)

Instrumento con forma de pala pequeña utilizado para tomar pequeñas cantidades de sustancias sólidas en polvo o granuladas.

Pinza (Tongs)

Instrumento metálico utilizado para sujetar objetos calientes o manipular sustancias sin contacto directo con las manos.

Gradilla (Test Tube Rack)

Soporte con orificios utilizado para mantener tubos de ensayo en posición vertical y organizados.

Soporte (Stand)

Estructura metálica utilizada para sostener otros materiales de laboratorio como pinzas, aros o buretas durante los experimentos.

Mechero (Bunsen Burner)

Dispositivo que produce una llama caliente utilizada para calentar, esterilizar o combustionar sustancias en el laboratorio.

Trípode o aro (Tripod)

Soporte de tres patas utilizado para sostener recipientes sobre el mechero durante el calentamiento de sustancias.

Mortero (Mortar)

Recipiente de porcelana, vidrio o mármol utilizado junto con una mano (pilón) para triturar, moler o mezclar sustancias sólidas.

Embudo (Funnel)

Instrumento cónico utilizado para transferir líquidos de un recipiente a otro sin derramamientos, especialmente a recipientes de boca estrecha.

Varilla (Stirring Rod)

Vara de vidrio utilizada para agitar mezclas y disoluciones, facilitando la homogeneización de las sustancias.

Rejilla (Wire Gauze)

Malla metálica con centro de amianto o cerámica utilizada para distribuir uniformemente el calor cuando se calientan recipientes de vidrio.

Embudo de decantación (Separatory Funnel)

Recipiente cónico con una llave en la parte inferior utilizado para separar líquidos inmiscibles por diferencia de densidades.

Pipeta (Pipette)

Instrumento de vidrio o plástico utilizado para medir y transferir volúmenes precisos de líquido de un recipiente a otro.

Escobilla (Test Tube Brush)

Cepillo con cerdas utilizado para limpiar el interior de tubos de ensayo y otros recipientes de laboratorio de difícil acceso.

Cubreobjetos (Cover Slip)

Pequeña lámina de vidrio muy delgada utilizada en microscopía para cubrir muestras colocadas sobre un portaobjetos.

Portaobjetos (Microscope Slide)

Pequeña placa de vidrio rectangular utilizada en microscopía para colocar muestras que serán observadas al microscopio.

Guantes (Gloves)

Protección para las manos utilizada para manipular sustancias químicas, materiales calientes o muestras biológicas.

Mechero de alcohol (Alcohol Burner)

Dispositivo que utiliza alcohol como combustible para producir una llama utilizada en calentamientos de baja temperatura.

Microscopio (Microscope)

Instrumento óptico que utiliza lentes para amplificar imágenes de objetos muy pequeños, permitiendo observar detalles invisibles a simple vista.

Chupón (Pipette Bulb)

Dispositivo de goma utilizado para succionar líquidos con una pipeta, evitando el contacto directo con la boca.

Placa calefactora (Hot Plate)

Dispositivo eléctrico con superficie plana que genera calor, utilizado para calentar recipientes de forma segura y controlada.

1.4 Material de seguridad

Gafas de protección (Safety Goggles)

Protección ocular que cubre completamente los ojos y el área circundante, protegiendo frente a salpicaduras químicas, vapores y partículas.

ETIQUETADO

Símbolos de seguridad

Tóxico

Puede causar daños graves o la muerte si se inhala, ingiere o entra en contacto con la piel.

Inflamable

Se enciende con facilidad al contacto con una chispa o calor.

Corrosivo

Puede destruir tejidos vivos y materiales al entrar en contacto con ellos.

Irritante

Puede causar irritación en la piel, ojos o tracto respiratorio.

Peligroso para el medioambiente

Puede causar daños al medio ambiente acuático, terrestre o a la capa de ozono.

Explosivo

Puede explotar bajo ciertas condiciones de presión, temperatura o impacto.

Productos de casa con símbolos de seguridad

Lejía

Símbolo: Corrosivo

Significado: Puede causar quemaduras graves en la piel y daños oculares irreversibles.

Amoníaco

Símbolo: Irritante

Significado: Puede causar irritación en las vías respiratorias y en los ojos.

Alcohol de quemar

Símbolo: Inflamable

Significado: Se enciende fácilmente en contacto con fuentes de ignición como llamas o chispas.
