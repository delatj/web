Cómo funciona el programa

Umbral y Segundos para pitar

Define el nivel de ruido máximo permitido (0 a 100). Si la clase supera este nivel de forma constante durante los "Segundos para pitar", sonará la alarma.

Avisos para límite y Contador

Cada vez que suena la alarma, sube el contador y el vaso se va llenando. Si se alcanza el "Avisos para límite" (ej. 10 veces), saltará un mensaje parpadeante en pantalla.

Modo Silencio (Corrección)

Al pulsarlo, te preguntará cuántos minutos de corrección quieres aplicar y activará una cuenta atrás. Durante este tiempo la clase debe estar callada. Si hacen ruido (superan el umbral) durante más de 2 segundos, el temporizador se detiene, avisa del fallo y te obliga a reiniciarlo desde cero con el botón rojo.
